---
title: Droppable
---

`Droppable` can be used to receive [`Draggable`](/components/draggable/) components.

```vue {2,6-11,20}
<script setup lang="ts">
import { Droppable } from "vue-simple-dnd"
</script>

<template>
    <Droppable
        v-slot="{ hovered }"
        @drop="(data) => {
            console.log(data);
        }"
    >
        <div
            class="flex gap-2 items-center mt-4 w-full bg-gray-800 h-32 border-2"
            :class="[
                hovered ? 'border-dotted border-red-500' : 'border-transparent',
            ]"
        >
            <!-- Draggable in here -->
        </div>
    </Droppable>
</template>
```
